# تطبيق المتجر (Flutter)

تطبيق واحد بواجهتين: **عميل** (متجر أزرق) و**تاجر** (لوحة متابعة)، حسب رتبة المستخدم بعد تسجيل الدخول.

## التشغيل

```bash
cd store_app
flutter pub get
flutter run
```

## الإعداد

- **رابط السيرفر**: عدّل `lib/config.dart` — للويب: `baseUrlWeb`، للمحاكي: `baseUrlAndroid`، للهاتف الحقيقي: `baseUrlOverride = 'http://IP_الكمبيوتر:3000'`.
- **مفاتيح API**: من لوحة التحكم → الإعدادات → مفاتيح API (صلاحية قراءة وكتابة)، انسخ Consumer Key و Consumer Secret إلى `lib/config.dart`.

## حسابات الاختبار

بعد أول تشغيل للسيرفر يُنشَأ تلقائياً:

| الإيميل             | كلمة المرور | الرتبة |
|---------------------|-------------|--------|
| customer@test.com   | 123456      | عميل   |
| merchant@test.com   | 123456      | تاجر   |

يمكنك أيضاً إنشاء مستخدمين من لوحة التحكم → **مستخدمو التطبيق**.

## المسار

1. فتح التطبيق → إن وُجد توكن صالح يتم التوجيه حسب الرتبة.
2. إن لم يوجد توكن → **شاشة الدخول** (إيميل + كلمة مرور؛ جوجل/أبل لاحقاً).
3. بعد الدخول: **عميل** → واجهة المتجر (#06a3e7)، **تاجر** → لوحة الطلبات والإحصائيات وتحويل الطلب.
